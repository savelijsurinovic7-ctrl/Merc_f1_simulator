# Merc_f1_simulator

My first Formula 1 simulator in Python. The track is Monaco-style — twisty, with variable width and red/white curbs.

## 🎮 Controls

| Key | Action |
|-----|--------|
| ⬆️ | Accelerate |
| ⬇️ | Brake / Reverse |
| ⬅️ | Steer left |
| ➡️ | Steer right |

## 🏁 Rules

- Complete 13 laps
- Maximum 3 crashes allowed
- 3 crashes = Game Over

## 🚀 How to run

1. Install Python 3.8+
2. Download `Mercedes simulator Python code.py`
3. Run in terminal:
   ```bash
   python Mercedes simulator Python code.py
